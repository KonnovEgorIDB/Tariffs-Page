# Tariffs Page - Frontend Test Task

## О проекте
Реализация страницы выбора тарифов для фитнес-сервиса по предоставленному Figma-макету.

## Стек технологий
- **Next.js 14/15** (App Router)
- **React**
- **TypeScript** (строгая типизация)
- **Tailwind CSS v4** (для адаптивной и Pixel Perfect верстки)

## Реализованный функционал
- 📥 **Асинхронная загрузка данных:** Тарифы динамически подтягиваются с предоставленного API.
- ⏱ **Интерактивный таймер:** Таймер обратного отсчета (2 минуты). При достижении 30 секунд цифры начинают моргать оранжево-красным.
- 💳 **Анимация цен:** При истечении таймера скидочные цены плавно исчезают (`opacity`), а базовые цены увеличиваются и встают на их место без резких скачков.
- 🎨 **Pixel Perfect адаптив:** Сложная сетка с абсолютным позиционированием бейджей (скидки "разрывают" рамку). Полностью адаптировано под мобильные устройства.
- ☑️ **Валидация формы:** Кнопка "Купить" пульсирует оранжевым при согласии. Если нажать без галочки — чекбокс моргает красным.
- 🏗 **Чистая архитектура:** Код разбит на переиспользуемые UI компоненты. Стейт-менеджмент изолирован от рендера карточек.

## Запуск проекта

1. Установите зависимости:
\`\`\`bash
npm install
\`\`\`

2. Запустите сервер разработки:
\`\`\`bash
npm run dev
\`\`\`

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере.
